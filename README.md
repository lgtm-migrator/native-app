# native-app

It is a react-native application for [thevaccinetracker](https://thevaccinetracker.com/)<br>
Developed using [expo](https://expo.io/) 

## Setup

```bash
git clone https://github.com/thevaccinetracker/native-app

npm i 
```
## Run Project

```bash
expo start
```
## Build Project 

For testing you application locally you can use following commands
```bash
expo build:android
expo build:ios
```
